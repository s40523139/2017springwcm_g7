下載這學期用的tiny2017_1GB 並打開start.bat 在tmp目錄下有一個CMsimfly的檔案

先查 ip用 ipconfig     

用 scite 把wsgi.py 打開 把 ip 改成自己的

步驟 : cd tmp -> cd CMsimfly -> pyhton wsgi.py 

打開leo 把wsgi.py的檔案打開 可以修改CMsimfly的內容
